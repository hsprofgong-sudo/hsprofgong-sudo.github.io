# Jinu Gong — Research Reviews

Personal homepage for Jinu Gong (Asst. Prof. @ Hansung Univ.). Static site reviewing my published papers.

## Pages
- `index.html` — profile + recent / most-cited highlights
- `recent.html` — three most recent papers
- `highlights.html` — three most-cited papers
- `about.html` — bio, research topics
- `archive.html` — full publication list
- `papers/` — individual paper review pages
- `assets/style.css` — academic/classic stylesheet
